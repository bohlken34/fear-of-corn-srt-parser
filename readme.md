To install, run:

`npm install -g fear-of-corn-srt-parser`

To run:

`foc-sp [command] <options>`

for example:
```
foc-sp version
foc-sp help

foc-sp parse <options>
      
      --file, -f ......... the file to parse
      --all, -a .......... parse all files in folder
      --recursive, -r .... used in combination with --all; include subfolders
```
